# TrackSpace
A full-stack task management platform designed to support both personal productivity and collaborative team workflows. Users can manage personal tasks within private workspaces or create team workspaces with projects, task assignments, and role-based access control.

## Project Vision
Modern professionals often use separate tools for:
- Personal task tracking  
- Team project management  
- Workflow monitoring  

TrackSpace aims to unify these needs into a single system by providing:
- Personal workspaces for private task management  
- Team workspaces for collaborative projects  
- Structured role-based access control  
- Scalable backend architecture

## Core Concepts
### Personal Workspace
Each user has a private workspace where they can:
- Create personal tasks
- Organize tasks by status
- Track progress independently
- Plan and organize daily activities

### Team Workspace
Users can create team workspaces and:
- Invite members
- Assign roles (ADMIN / MEMBER)
- Create projects
- Assign tasks to team members
- Monitor team progress

## Features
### Authentication & Security

### Workspace Management
- Create personal and team workspaces
- Invite users to team workspace
